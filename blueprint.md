# Project Blueprint: Face Age Estimation Web Application

## Overview

이 프로젝트는 사용자가 얼굴 이미지를 제공하면 해당 이미지의 인물 나이를 추정하는 웹 애플리케이션을 만드는 것을 목표로 합니다. 이전 버전에서는 파일 업로드를 사용했지만, 이제는 사용자의 웹캠을 통해 이미지를 직접 촬영하는 기능과 더 현대적이고 시각적으로 매력적인 디자인을 추가할 것입니다. 나이 추정 자체는 머신러닝 모델 또는 외부 API가 필요하며, 이 단계에서는 시뮬레이션된 결과를 제공하고 추후 통합될 예정입니다.

## Current State

*   기본 HTML, CSS, JavaScript 파일(`index.html`, `style.css`, `main.js`)이 존재합니다.
*   프로젝트는 프레임워크를 사용하지 않는 웹 구조를 가집니다.
*   초기 버전에서는 파일 업로드 및 이미지 표시, 나이 추정 시뮬레이션 기능이 구현되었습니다.

## Implemented Features (Updated Version)

*   **향상된 사용자 인터페이스**:
    *   웹캠을 통해 이미지를 직접 촬영하는 기능.
    *   촬영된 이미지를 표시하는 영역.
    *   나이 추정 결과를 표시하는 영역.
*   **현대적인 스타일링**: 시각적으로 매력적이고 반응형이며, 사용자 경험을 향상시키는 CSS 스타일.
*   **카메라 제어**: JavaScript를 사용하여 웹캠에 접근하고, 비디오 스트림을 표시하며, 이미지 캡처를 처리합니다.
*   **이미지 처리**: 캡처된 이미지를 데이터 URL로 변환하여 나이 추정 로직에 전달합니다.

## Plan for Current Request (Design Improvement and Camera Capture)

현재 요청(디자인 개선 및 카메라 직접 촬영)을 위해 다음 단계를 수행합니다.

1.  **`blueprint.md` 업데이트**: 이 문서를 현재 변경 사항과 계획을 반영하여 업데이트합니다. (완료됨)

2.  **`index.html` 수정**:
    *   기존 파일 업로드 입력 필드를 제거하고, 웹캠 스트림을 표시할 `<video>` 요소와 이미지 촬영을 위한 `<button>` 요소를 추가합니다.
    *   촬영된 이미지를 표시할 `<img>` 요소와 나이 추정 결과를 표시할 `<div id="ageResult">`는 유지합니다.
    *   전반적인 레이아웃 및 마크업을 조정하여 새로운 디자인과 카메라 기능에 맞춥니다.

3.  **`style.css` 수정**:
    *   `body` 및 `.container`에 대한 스타일을 개선하여 더 현대적이고 매력적인 디자인을 만듭니다.
    *   `<video>` 요소, 캡처 버튼, 그리고 이미지 미리보기 영역에 대한 스타일을 추가하여 시각적 일관성을 유지합니다.
    *   CSS 변수, 쉐도우, 애니메이션 등 최신 CSS 기능을 활용하여 디자인을 향상시키고 사용자에게 더 나은 피드백을 제공합니다.
    *   반응형 디자인을 보장합니다.

4.  **`main.js` 수정**:
    *   `DOMContentLoaded` 이벤트 리스너 내에서 필요한 DOM 요소들을 가져옵니다.
    *   `navigator.mediaDevices.getUserMedia()`를 사용하여 웹캠 스트림에 접근하고, 이를 `<video>` 요소에 연결합니다.
    *   카메라 캡처 버튼에 클릭 이벤트 리스너를 추가합니다. 이 리스너는 `<canvas>` 요소를 사용하여 비디오 프레임을 캡처하고, 캡처된 이미지를 `<img>` 요소에 표시하며, 나이 추정 함수에 전달합니다.
    *   `estimateAge(imageData)` 함수를 업데이트하여 캡처된 이미지 데이터를 처리할 수 있도록 합니다. (현재는 시뮬레이션된 결과를 제공)
    *   카메라 접근 권한 요청 및 오류 처리 로직을 추가합니다.

5.  **나이 추정 로직 통합 (추후)**:
    *   현재는 시뮬레이션된 나이 추정 결과를 제공합니다.
    *   실제 나이 추정을 위해서는 외부 API(예: Google Cloud Vision API, Azure Cognitive Services) 또는 클라이언트 측 머신러닝 모델(예: TensorFlow.js) 통합이 필요합니다.
    *   이러한 통합에 대한 논의는 이 단계 이후에 진행합니다.

6.  **정제 및 테스트**:
    *   모든 요소가 기능적으로 작동하는지 확인하고 UI가 반응형인지 테스트합니다.
    *   카메라 접근, 이미지 캡처, 이미지 표시 기능을 테스트합니다.
    *   사용자에게 웹사이트 사용 방법과 나이 추정 기능의 현재 한계에 대해 명확하게 안내합니다.
